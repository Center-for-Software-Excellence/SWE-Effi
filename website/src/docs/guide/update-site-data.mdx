### Leaderboard

##### Table 1

Append new records to `website/src/config/leaderboard/data.json`, each entry should look like:

```json
{
  "scaffold": "Agentless-mini",
  "model": "Qwen3-32B",
  "total": 7438,
  "cpuTime": 0.76,
  "inputToken": 34.1,
  "outputToken": 9.31,
  "calls": 2,
  "infTime": 7437,
  "resolveRate": 28,
  "precision": 48.3
}
```

##### Table 2

```json
{
  "scaffold": "Agentless-mini",
  "model": "Qwen3-32B",
  "avgTotalTimeU": 7438.0,
  "avgTotalTimeR": 7438.0,
  "avgCPUTimeU": 0.76,
```

### Chart pipeline

1. **Add raw stats**  
    Move new benchmark outputs to
   `benchmark/results/agent-scaffold-stats/<scaffold-name>/<model-name>-combined_stats.json`

2. **Copy them into raw data folder** (run from repo root)

```bash
# copies & renames files into website/public/data/benchmark/raw/
python scripts/transform_benchmark.py
```

3. **Generarte chart-ready data**

```bash
cd website
# writes to website/public/data/chart/
npm run generate-data
```
